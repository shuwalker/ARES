# Gumroad listing — Obsidian Cosmos

## Your Obsidian graph, but it looks like a video game

Turn your Obsidian vault into a visual dashboard — a single HTML file you open locally, no server, no account, no API.

---

### WHAT THIS IS

Obsidian Cosmos reads your `.md` files and renders them as a live graph in your browser.

- **Solar System theme** — folders become orbital rings, files are glowing planets, `[[wikilinks]]` become gravity arcs.
- **Skill Tree theme** — folders become branches, files become skill nodes, links become parent-child edges.
- **Theme switcher** — hot-swap via UI or `Ctrl/Cmd + T`.
- **Single file** — just `cosmos.html`, no build step, works offline.

### HOW TO USE

1. Select your vault root folder from the **Open vault…** button.
2. Wait a few seconds while notes are parsed.
3. Switch themes with the **Theme** button or `Ctrl + T`.
4. Screenshot the result and share.
