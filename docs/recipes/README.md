# ARES Recipes

Reusable build recipes extracted from the Building Ares workflow.

These are not private runtime configs. They are patterns other builders can copy, adapt, and turn into their own tools.

## Recipes

- [YouTube Playlist Research Pipeline](youtube-playlist-research-pipeline.md) — turn a watch-later/research playlist into reproducible build notes.
- [Video Production Workflow](video-production-workflow.md) — run an episode from idea capture to publishable package without hiding work in one person's head.

## Recipe standard

Each recipe should include:

1. the problem it solves;
2. the inputs and outputs;
3. the minimum data model;
4. the operator loop;
5. failure modes;
6. verification gates;
7. what is safe to automate and what requires approval.
